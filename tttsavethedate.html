How to Use This Page

1. Preview the Video Card

The video preview shows the Save The Date video invitation your guests will receive. Review it before sending anything.

2. Review Your Guest List

The guest table displays the people available to receive your Save The Date message. Check names and email addresses before sending.

3. Select Guests

Use the checkboxes to select individual guests or use the master checkbox to select the full list at once.

4. Send by Email

Click Send Email to open the composer, review the recipients, edit the subject and message, and send the Save The Date invitation.

5. Customize the Message

You can adjust the email text before sending, while the personalized Save The Date link is added automatically.

6. Use Personalized Links

Each selected guest receives their own unique Save The Date link connected to their guest record.

7. Send by SMS

Use Send SMS to generate a QR code that can be scanned with your phone for quick mobile sharing.

8. Track Sending Progress

When emails are being sent, the progress window shows how many messages have been processed. Do not close the page until sending is complete.

9. Check Delivery Issues

If any message fails to send, the affected guest row is highlighted so you can review and resend if needed.

10. Update Guest Details

Guest names and email addresses can be edited directly in the table before sending your Save The Date message.

11. Preview Guest Experience

Use the preview options to check how the Save The Date experience will appear to guests.

12. Send Early Announcements

Use this page before RSVP to give guests the first official notice of your wedding date and celebration.

Important Notes

i
Save The Date is designed for early wedding announcements, not for collecting RSVP responses.
i
Select at least one guest before sending Email or SMS notifications.
i
Each guest receives a personalized link, so avoid copying one guest’s link and sending it manually to another guest.
i
If a delivery issue appears, check the highlighted guest row and confirm the email address before resending.

In short: preview your Save The Date video, select guests, send personalized invitations by email or SMS, track delivery, and notify guests before the RSVP stage begins.
