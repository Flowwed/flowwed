Updates

i
The Updates page helps you share important wedding announcements with your guests. Use Updates whenever plans change or new details become available. It keeps everyone up to date with your wedding-related news. Once an update is published, guests can view it on your Updates webpage, and you can notify them by email or SMS.

How to Use This Page

1. Show or Hide Updates

Use the visibility checkbox to control whether the Updates page is visible to guests. Keep it hidden while preparing content if needed.

2. Write an Update

Use the text editor to write an announcement, reminder, schedule change, or any message your guests should know.

3. Add It to the Update Feed

When saved, your update is added to the feed with the current date so guests can see the latest announcements.

4. Save Text Changes

Click Save Text Changes to publish your latest update to the Updates page.

5. Preview Guest View

Use Preview Your Page Updates to see exactly how your Updates page will appear to guests.

6. Notify by Email

After publishing an update, use Send Email to notify guests that new information is available.

7. Notify by SMS

Use Send SMS to generate a QR code and quickly share the update link from your phone.

8. Delete Old Updates

Remove updates that are no longer needed. A confirmation window helps prevent accidental deletion.

Important Notes

i
Updates is designed for announcements and information changes, not for collecting RSVP responses.
i
Always save your text changes before leaving the page.
i
If the Updates page is hidden, guests will not see the updates until you make the page visible again.
i
Use Email or SMS after saving an update if guests need to be notified immediately.
