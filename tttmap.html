Travel & Event

i
The Travel & Event page helps you organize the practical information your guests need before the wedding. You can build a wedding-day timeline, add travel and hotel details, provide venue information, create FAQs, preview the guest page, and send updates by email or SMS.

How to Use This Page

1. Show or Hide This Page

Use the visibility checkbox to control whether the Travel & Event page is visible to guests. Keep it hidden while you are editing, then turn it on when the information is ready.

2. Build the Event Schedule

Add the main wedding events such as ceremony, cocktail hour, reception dinner, and after-party. Each event can have its own title and time.

3. Add More Wedding Events

Use “Add wedding event” to include extra activities such as welcome drinks, rehearsal dinner, brunch, excursions, or any custom event guests should know about.

4. Add Travel Information

Fill in airport details, transfer time, travel notes, arrival tips, and any helpful instructions for out-of-town or destination wedding guests.

5. Recommend Hotels

Add recommended hotels, accommodation notes, and booking links. This helps guests quickly understand where they can stay near the wedding location.

6. Set Venue Details

Enter the venue name, address, and venue note. The map link is automatically generated from the venue information so guests can open directions easily.

7. Configure Dress Code

Choose a dress code such as formal, cocktail, semi-formal, or casual. You can also edit the description to match the style of your wedding.

8. Add Important Information

Use the preset options for parking, outdoor wedding, adults-only celebration, or unplugged ceremony. You can also add your own custom notes for guests.

9. Add Weather & Local Links

Add weather links, local travel resources, booking platforms, and things to do so guests can plan their stay more comfortably.

10. Build the FAQ Section

Use FAQ presets for dress code, kids, plus-one policy, parking, arrival time, and meal selection. You can also add extra custom questions.

11. Save Your Updates

Click Save Travel & Event after editing. The page stores your timeline, venue details, travel information, FAQ, and guest instructions.

12. Preview the Guest Page

Click Preview your webpage to see exactly how the Travel & Event page will appear to guests before you send updates or make the page visible.

13. Send Email Updates

Use the guest table to select individual guests or select all guests. Click Send Email to open the composer. The Travel & Event page link is added automatically, so guests can view the latest event and travel information.

14. Send SMS Notifications

Use Send SMS to generate a QR code. Scan it with your phone to quickly send travel reminders or important event updates to selected guests.

Important Notes

i
Travel & Event is especially important for destination weddings, out-of-town guests, multi-day celebrations, or weddings with several scheduled events.
i
Always preview the page after editing. Check event times, hotel links, venue address, map link, dress code, FAQ answers, and important notes before sending updates.
i
If plans change, update the page first, save it, preview it, and then notify selected guests through email or SMS.
i
Keep this page hidden while information is incomplete. Make it visible only when guests are ready to view the final or updated travel details.
