Seating Chart

i
The Seating Chart page helps you organize where each guest will sit during your wedding celebration. Create tables, assign guests, manage plus-ones, monitor table capacity, and export your final seating plan from one place.

How to Use This Page

1. Create Tables

Three ruby-colored tables are displayed in the seating chart preview when the page opens. You can create additional tables or remove any tables that are not needed.

2. Set Seat Capacity

Adjust the number of seats available at each ruby-colored table using the seat controls.

3. Review RSVP Guests

Confirmed and maybe guests are automatically available for seating assignments.

4. Filter Guest Lists

Use the filter to quickly switch between All, Confirmed, and Maybe guests for easier seating planning.

5. Assign Guests to Tables

Assign guests to a table using grey table selection buttons in the "Assign Table" column. Once a guest is assigned, their name will appear on the corresponding ruby-colored table in the seating chart preview.

6. Manage Plus-Ones

The system automatically counts additional guests, known as "plus ones", accompanying the main guest and reserves the required number of seats for the entire party at the same table.

7. Move Guests Between Tables

Guests can be reassigned to another table at any time. Simply click "Unassign Guest" and then select a new table in the "Assign Table" column.

8. Prevent Overbooking

If a table exceeds its seating capacity, the system will immediately display a warning. To proceed, either increase the number of seats at that table using the seat controls or assign the guest to another table with available seating.

9. Monitor Remaining Seats

Each table displays its remaining capacity, making it easier to balance guests across all tables.

10. Email Seating Updates

Email the latest seating plan directly to planners, venues, vendors, or other event stakeholders.

11. Export & Print Seating Plan

Download the seating plan as a CSV file and print it for your records or event coordination.

12. Automatic Saving

There is no Save button. All seating changes are saved automatically.

Important Notes

i
Review your seating plan carefully before exporting or sharing it with vendors.
i
