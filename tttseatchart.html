How to Use This Page

1. Create Tables

Create new tables or remove tables that are no longer needed.

2. Set Seat Capacity

Adjust the number of seats available at each table using the seat controls.

3. Review RSVP Guests

Confirmed and maybe guests are automatically available for seating assignments.

4. Filter Guest Lists

Switch between All, Confirmed, and Maybe guests for easier planning.

5. Assign Guests to Tables

Assign guests to a table with one click using the table selection buttons.

6. Manage Plus-Ones

The system automatically counts additional guests and reserves the required seats.

7. Move Guests Between Tables

Guests can be reassigned to another table at any time.

8. Prevent Overbooking

If a table exceeds its capacity, the system warns you immediately.

9. Review Table Occupancy

See how many guests are assigned and how many seats remain available.

10. Expand Large Tables

Use Show All to display the full guest list for tables containing many guests.

11. Email Seating Updates

Send seating-related updates directly to selected guests.

12. Export & Print Seating Plan

Download the seating plan as a CSV file for planners, venues, and vendors.

13. Automatic Saving

Your seating layout is automatically saved while you work.

14. Monitor Remaining Seats

Each table shows remaining capacity so you can balance guest distribution efficiently.

Important Notes

i
Seating capacity includes all confirmed guests and their registered plus-ones.
i
The system prevents assigning more guests than a table can accommodate.
i
Review your seating plan carefully before exporting or sharing it with vendors.
i
Seating assignments can be updated at any time before the wedding.

In short: create tables, assign guests, manage seating capacity, accommodate plus-ones, export reports, and organize your complete wedding seating plan from one dashboard.
